# 🩺 AI Diabetic Lifestyle Optimizer

An AI-powered system that predicts short-term glucose levels and gives safe lifestyle suggestions for diabetic users.

## 📦 Project Structure

diabetic-optimizer/
├── backend/ # FastAPI backend
│ └── app/ # Main app files (predictor, recommend, etc.)
├── simulator/ # Synthetic data generator & simulated environment
├── data/ # Datasets (.csv)
├── models/ # Trained ML models (.pt)
├── notebooks/ # ML experiments / Jupyter notebooks
├── frontend/ # React-based frontend (later)
├── requirements.txt # All dependencies
└── README.md
